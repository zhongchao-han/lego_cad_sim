# LEGO CAD Simulation System - 产品文档

## 1. 项目概览
本系统是一个高度模块化的乐高 CAD 仿真系统，通过强类型接口定义实现精准组装。

## 2. 核心架构设计
- **树状组合模式**: `Interface` -> `Port` -> `Part` -> `Edge` -> `Assembly`。
- **ROOT 动态迁移**: 始终将当前操作的目标点定义为物理地基，确保坐标系实时对齐。

## 3. 关键特性

### 3.1 强类型接口配合
- **查表法配合判定**: 自动识别间隙/摩擦/过盈配合。

### 3.2 自动运动学推导与约束
- **过约束合并**: 识别平行销钉并将其退化为 `Fixed Joint`。
- **刚性相容性校验**: 强制拦截违反几何规律的孔距对齐。

### 3.3 交互逻辑核心 (The Interactive Soul)
- **先选即动与点谁谁走**: 建立在“用户意图第一”原则之上的确定性位移逻辑。
- **步进式轴向拆解**: 拆解多连接点时，零件执行“向外弹出”动画，模拟真实的拔插触感。
- **自动齿隙咬合 (Auto-Gear Meshing)**: 拼装时系统自动微调齿轮相位以实现即插即用的咬合，极大地减少了手动对齐的工作量。

### 3.4 沉浸式装配工作流 (The Focused Workflow)
- **焦点预览**: 零件进入画布前处于放大预览态，支持 360° 全方位选点，避免遮挡。
- **智能空间分区**: 引入“暂存工作台”管理拆解成果，引入“物料回收”保持场景整洁。

### 3.5 精度保障 (Alignment Precision)
- **stripAxis 投影**: 强制将所有 Snap 路径投影至零件的几何中心轴，实现亚毫米级的对齐精度。

### 3.6 交互细节增强 (Interaction Polish)
- **摄像机自适应对焦 (Adaptive Camera Focusing)**: 选中端口后摄像机自动平滑移动至操作点，配合基于光标的缩放逻辑 (`Dolly to Cursor`)，实现类似专业 CAD 的微操作体验。
- **单一责任摄像控制 (CameraController Pattern)**: 摄像机控制逻辑与业务模块解耦，确保装配与复核场景具备完全一致的过渡动效与稳定度。

## 4. 模块职责划分
- `topology_manager.py`: 负责装配拓扑解算、树/环关系维护。
- `port_library.py`: 零件语义库加载器（原 `ldraw_parser.py`），系统唯一的端口真理来源。
- `port_semantics.py`: 定义物理接口配合（Fit）、极性（Gender）与截面（Profile）语义。
- `core_constants.py`: 统一定义 LDU (0.0004m) 等物理常数。
- `frontend/`: 基于 React + Three.js 的交互环境。
