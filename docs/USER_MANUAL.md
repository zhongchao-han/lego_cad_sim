# LDraw Web CAD 模拟器 —— 创作者使用手册 (User Manual)

欢迎使用 LDraw Web CAD 模拟器！本手册将带您快速掌握这款拥有 Studio 2.0 原生极致操作手感的专业全息积木搭建工作台。

---

## 1. 基础视口操控与浏览
* **旋转视角**：在空白处按住 `鼠标左键` 或 `鼠标右键` 拖拽，可自由环视3D空间。
  * *MacBook 触控板*：直接单指按下拖动（或重压拖放）。
* **平移视角**：按住 `Shift + 鼠标左键` 拖动，或直接按住 `鼠标中键(滚轮)` 拖拽，平行位移镜头重心。
  * *MacBook 触控板*：按住键盘 `Shift` + 单指按下拖动，或直接使用**双指轻扫滑动**。
* **拉近/缩放**：滚动 `鼠标滚轮` 调节远近距离。
  * *MacBook 触控板*：自由使用**双指捏合 / 张开** (Pinch to zoom) 手势。

---

## 2. 积木交互与选择 (Selection)
系统支持基于逻辑分组和个体的多层级钻取选择。在极简的“盲操（Blind-Click）”范式下，积木表面平时保持绝对干净，端口指示器（Gizmo）完全隐形。一旦你选中了零件，它身上所有的物理孔洞和插销端口才会像孔雀开屏般全部亮起常驻，实现**所见即所得**的精准对齐。

* **单体选中 (INDIVIDUAL)**：直接 `左键单击` 积木，可精准聚焦到该单块积木（外围呈现白色光圈，同时**解禁所有端口指示器**）。左键单击永远只负责单选，绝不误触。
* **连体组全选 (GROUP)**：使用 `极速双击 (Double-Click)` 零件，系统会通过物理约束连结网，瞬间全选与它相连的整个模型组合体。
* **追加多选**：按住 `Shift` 或 `Cmd/Ctrl` 键时，接连点击多块积木可将它们全部裹挟加入您的控制组，再次点击已加入的积木则是减选。
* **空拍反选**：点击场景任意**深色虚无背景底板**，即可放弃/解除当前全部焦点。

## 3. 高能装配命令 (Keyboard Shortcuts)
系统全面切断了对复杂2D按钮面板的依赖，提倡全盲操级别的肌肉记忆：

| 操作种类 | 快捷键 | 使用说明 |
| :--- | :--- | :--- |
| **复制与拾取** | `Cmd/Ctrl + C` | 暂存所选项进系统剪贴板。 |
| **幽灵跟手粘贴** | `Cmd/Ctrl + V` | **核心神技**！按下后克隆体积木会像“幽灵”般漂浮跟随您的鼠标准星，并对地形产生重力爬坡侦测。找到称心如意的地方后**单击鼠标左键**即可下放入库。（在此期间按下 `Esc` 或右键则取消放置）。 |
| **一步到位克隆** | `Cmd/Ctrl + D` | 直接完成 复制+粘滞态激活 的连招。 |
| **Alt影分身** | `按住 Alt + 点击零件` | 系统会把母体积木留在原位，产生一个全新克隆体进入“幽灵跟随”状态，点击即可放置。 |
| **上下文旋转** | `[` / `]` 或 `<` / `>` | 在选中零件时（或轴滑动期间），按下此键立即以 **90度** 的节拍硬核旋转该零件，盲选对齐结构，再无遮挡。 |
| **销毁组件** | `Delete` 或 `Backspace` | 将选中实体无情蒸发并斩断它们相关联的物理连接桥。 |
| **剥离透视** | `H` 键 | 将当前选中物体设为隐身（不参与物理碰撞挡道），常用于做“内部手术”。 |
| **取消伪装** | `Alt + H` 键 | 全景解禁，将世界上所有被您藏匿 (H) 的零件复显。 |
| **摄影机神移** | `F` 键 | 迷失于茫茫的底盘宇宙时，选中某个您留意的零件，按下 F 键，镜头一秒自动将目标无死角框进最佳视角的屏幕正中心。 |
| **强制时空纠错**| `Cmd/Ctrl + Z` | 时光倒流 (Undo)，打退一切失手的放置与删除（含所有物理绑定恢复）。 |
| **逆转未来** | `Cmd/Ctrl + Y` 或 `Shift+Ctrl+Z` | 还原纠错 (Redo)。 |

## 4. 极致装配技巧篇 (Pro-Tips)

### 🎯 **标准的“两段式”盲点装配流 (Click-to-Focus, Click-to-Act)**
面对密集的科技件插销阵列，请使用以下标准交互以达到 100% 的容错率：
1. **第一段 (聚焦 Focus)**：单击空闲零件的光滑表面。零件加框，其所有的可用端口指示器全面亮起，防误触的物理护盾同时升起拦截。
2. **第二段 (锁定 Act)**：看着亮起的某个端口箭头，将鼠标移过去再次单击。系统会瞬间吸收事件并锁定端口，进入 `SOURCE_LOCKED` 准备连接。

### 🚀 **强力拼装（Shift无视装甲降维法）**
在利用盲按操作 (Blind-Click) 进行 **轴向深切入滑动 (Axial Sliding)** 插入十字轴的长短时，系统为了模拟真实的积木抗拒性存在默认的防穿模阈值。 
如果您正在发挥超常创造力，请在使用方向键滑动期间**狠狠按住 Shift 键**，所有的物理边界将灰飞烟灭，您可以强制让模型之间交错互插至极负点。

### ♾️ **图章模式（连续无缝拼接）**
在左侧零件库预览窗选中零件的端口后，系统会自动进入“图章”模式：
1. **一触即发**：点击主舞台的目标端口，系统会在瞬间替你完成吸附与对齐锁定。
2. **无需确认**：放好前一个零件后，**不需要按任何键**提交。直接移动鼠标，准星指向下一个插槽，你会立刻看到下一个零件的“幽灵预览”。
3. **点哪贴哪**：再次点击，瞬间克隆并吸附。想要退出这种疯狂的流水线涂抹模式？随时按下 `Esc` 键或单击鼠标右键即可优雅退场。

### 🧩 **使用部件搜索图鉴建立您的首个单元**
1. 屏幕左侧具有全智能极速 AI 文字搜索引擎支持 (*MeiliSearch*)。
2. 在搜索框键入如 `Technic Axle 2 Red` 等字样。 
3. 在左列结果中敲击条目。
4. 模型会瞬时降落在全息检查台进行 360 度体检。当您满意时，在右侧验证通过它即会跃升加入正式组装场！

*(注：全键盘盲操特性同样受搜索框避让锁保护。在您打字聊天或检索零件时，即使按下含有 Delete、F 或 H 的文字都不会摧毁您的搭建心血，请放心输出)*
