# LDraw Web CAD 核心编辑器功能产品文档 (v1.3)

## 1. 产品概述
本模块聚焦于 Web 端三维搭建系统的「实体拓扑编辑」与「高频工程辅助交互」。通过键盘与状态流的融合，为用户提供堪比原生客户端（如 Studio 2.0）的极速搭建体验，支持所见即所得的零件层级结构修改与防呆回溯。

## 2. 核心功能矩阵 (Studio 2.0 Parity)

### 2.1 连续多选连结 (Multi-Selection)
* **适用场景**：想要同时操作多块零件（如更换颜色、整体删除或成套克隆）。
* **交互规则**：按住 `Shift` 或 `Cmd/Ctrl` 键时，再用鼠标点选零件，会将该零件累加进入当前的高亮选中池（或将其弹射取反移出选中池），做到连选免清空。

### 2.2 全局镜头瞬移归心 (Camera Autofocus)
* **适用场景**：大规模群落（超过2000积木）搭建中，镜头焦点脱离控制，想要立刻注视某些积木特写。
* **交互规则**：选中一个或多个零件，按下 `F` 键，相机会自动捕捉所选模型的**3D包围群集几何中心点**，并发动缓动飞行将其完美包围展示在屏幕正中央。

### 2.3 物理碰撞与吸附越权 (Shift-Override)
* **适用场景**：利用物理碰撞漏洞实现创意搭建，或是规避受限空间的死板插槽防呆。
* **交互规则**：在执行沿轴拖动（Axial Sliding）手势时，**死死按住 Shift 键**，所有的 +/- 碰撞限位将立即崩塌瓦解，允许您将积木拖到穿模的负空间里。

### 2.4 极速全息拽包 (Alt-Drag Clone)
* **适用场景**：在底盘上快速铺设成排的地板砖、轮毂等。
* **交互规则**：在即将沿轴拖动或刚点选积木时，**按住 Alt 键** 然后用鼠标拖走该零件。旧母体会滞留在原地，被拖脱出的是其全新生成的 UUID 复制品，且自动切换焦点。

### 2.5 真实 3D 鼠标吸附抓取 (Cursor-Following Paste)
* **适用场景**：原有的复制粘贴永远掉在固定偏移位太生硬，需要精确瞄孔部署。
* **交互规则**：
  * 执行 `Cmd/Ctrl + V`（或克隆）时，粘贴出来的物件不会立刻落地。
  * 它们将被打成一个相对坐标群组，**半透明化跟随着鼠标（射线的碰撞焦点）在地板或实体上自动顺位爬升或降落**。
  * 在寻找好目标坐标后，只需单压**鼠标左键**，实锤落子！

### 2.6 撤销与重生防线 (Undo / Redo)
* 按下 `Cmd/Ctrl + Z` 进行系统级步进回退；`Cmd/Ctrl + Y` 前进跨步。最多保留 50 步防伪防冲突栈记录。

### 2.7 连续拼接图章模式 (Continuous Stamp Mode)
* **适用场景**：快速大量布置同一种连接件（如插销、地砖）而无需反复从零件库中提取。
* **交互规则**：
  * 从零件库选中零件并在预览窗口点击源端口后，触发全局 **图章模式 (`continuousPlacementSource`)**。
  * 将其吸附到场景中的第一个目标端口后，系统会自动进入局部滑动微调状态。此时无需额外键入确认指令，鼠标直接移向下一个目标端口，依然会完美显示“幽灵网格”预览。
  * 再次点击目标端口，系统会在毫秒级内静默提交上一个零件的深度状态，并瞬时克隆一个相同的源零件放置在当前位置。
  * 随时按下 `ESC` 或右键即可打断循环操作并退出图章模式，回归 IDLE 态。

## 3. 防冲突体系与焦点安全锁
当光标聚焦在表单输入事件（如 MeiliSearch 输入、后台参数设置区）时，由于隔离层的屏蔽判断，所有快捷干预功能（如 Delete、H 等）静默阻尼失效。
