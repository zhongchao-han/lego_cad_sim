# LDraw Web CAD 核心编辑器功能产品文档 (v1.3)

## 1. 产品概述
本模块聚焦于 Web 端三维搭建系统的「实体拓扑编辑」与「高频工程辅助交互」。通过键盘与状态流的融合，为用户提供堪比原生客户端（如 Studio 2.0 / LDD）的极速搭建体验，支持所见即所得的零件层级结构修改与防呆回溯。

## 2. 核心功能矩阵

### 2.1 撤销与重做流 (Undo / Redo)
*   **适用场景**：用户发生误触删除、不理想的吸附贴近、选错主件时的纠错。
*   **交互规则**：
    *   按下 `Cmd/Ctrl + Z` 进行上一步历史状态回滚。
    *   按下 `Cmd/Ctrl + Shift + Z` (Mac) 或 `Cmd/Ctrl + Y` (Windows) 进行状态前进。
*   **业务极限**：最多回溯 50 步，超出后丢弃最早记录；当通过拖入/点击产生新的物理分支时，原有的 Redo 池将被自动清空以防止时间线撕裂。

### 2.2 剪贴板与复印流 (Clipboard)
*   **适用场景**：当拼装中大量重复结构（如两组对称引擎、三侧相同护甲面板）时的快速繁殖。
*   **交互规则**：
    *   `Cmd/Ctrl + C` (复制) -> 读取现有选择实体，暂存入全局只读剪贴板。
    *   `Cmd/Ctrl + V` (粘贴) -> 在当前世界坐标上释放内容，并在 X/Y/Z 三维均做 `+0.05` 微量偏移以保证视觉穿模抗拒（防止重叠看不见导致误操作），且全选焦点自动转移给新零件。
    *   `Cmd/Ctrl + D` (克隆) -> 即为上述 C+V 串通一气的一步到位操作。

### 2.3 销毁与阻断流 (Lifecycle)
*   **适用场景**：搭建偏序中的摒弃结构，或者中途需要重新思考布线。
*   **交互规则**：
    *   选中单零件/多连通组件后，按下 `Delete` / `Backspace` 销毁零件。系统同时严格销毁该实体上粘附的所有约束连结。
    *   运行中的动作可通过按下 `Esc` 键强行打断（清空所有悬浮幽灵与选择器）。

### 2.4 可视边界控制流 (Visibility)
*   **适用场景**：当拼装体非常繁杂导致内部关键连接点或暗扣被外部护板遮盖时，用于内视。
*   **交互规则**：
    *   `H` -> 隐藏当前高亮或全选中的零件。该零件从渲染池与光线投射碰撞树中双向剥离（可看穿，不可被选中，甚至不可被吸附）。
    *   `Alt + H` -> 解锁并显示三维空间中全部的隐藏实体。

## 3. 防冲突体系与焦点安全锁
当光标聚焦在：
- `MeiliSearch` 全局搜索栏 `<input>`
- 代码调控后台 `<textarea>` 
时，以上所有编辑态 `KeyDown` 事件**自动全部失效避让**，根治了打字途中引发场景撕裂的恶性体验痛点。
